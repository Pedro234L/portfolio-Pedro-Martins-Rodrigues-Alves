# 🌐 Batalha de Modelos e Engenharia de Prompt (XML)
 ## 📝 Descrição do Projeto
Este projeto documenta um experimento comparativo entre diversos modelos de linguagem (LLMs) utilizando a técnica de **Prompt Estruturado em XML**. O objetivo central foi a geração de uma página *Single Page* funcional sobre Finanças e Investimentos, avaliando a precisão técnica, criatividade e a eficiência de cada IA.

O experimento focou em como diferentes arquiteturas processam diretrizes de design, obrigatoriedades técnicas e integração de código (HTML5/CSS3) a partir de um protocolo de execução estruturado.

 ## 📊 Resultados e Aprendizados
O experimento revelou dados importantes sobre a capacidade de processamento das IAs testadas:

* **Domínio de Estrutura:** O modelo **Claude** demonstrou a maior compreensão da estrutura XML, entregando uma interface fluida, interativa e sem bugs.
* **Análise de Verbosidade:** Identificamos uma variação drástica no uso de tokens para o mesmo objetivo; enquanto o ChatGPT utilizou **1150 tokens**, o Claude gerou **6200 tokens** para garantir maior detalhamento.
* **Precisão Técnica:** Modelos como o DeepSeek e o Grok apresentaram alta criatividade, porém alguns apresentaram falhas em elementos específicos, como gráficos financeiros ou funções de botões.
* **Eficiência:** O estudo permitiu entender qual modelo é mais adequado para geração de código complexo e qual entrega resultados mais concisos.

 ## 🔧 Como Executar
1. Abra o arquivo `Trabalho_pagina.pdf` para copiar o protocolo de prompt em XML.
2. Cole o prompt em uma das ferramentas de IA testadas.
3. Salve o retorno como um arquivo `.html` e execute no navegador para visualizar a página de finanças gerada.
