# 🤖 Laboratório de Classificação Visual e Análise de Viés
 ## 📝 Descrição do Projeto
Este projeto consiste em um classificador de imagens desenvolvido para distinguir perfis de usuários (Fãs do Messi vs. Fãs do Cristiano Ronaldo) utilizando a plataforma **Teachable Machine**. O foco central do experimento foi observar como a IA utiliza estereótipos comuns e características visuais para realizar predições. 

O modelo foi treinado para identificar padrões sutis, alcançando um nível de confiança de **97%** na identificação das classes durante os testes em tempo real.

 ## 🚀 Tecnologias Utilizadas
* **Plataforma:** Teachable Machine (Google Cloud) 
* **Conceitos:** Visão Computacional, Aprendizado Supervisionado e Ética em IA. 
* **Ferramentas:** Captura via Webcam e análise de Redes Neurais. 

 ## 📊 Resultados e Aprendizados
* **Mecanismo de Viés:** Identificamos que a IA utiliza estereótipos e preferências comuns para distinguir as classes, o que levanta discussões importantes sobre como a máquina "enxerga" preferências humanas. 
* **Impacto Social:** Analisamos que, embora o impacto profissional seja nulo neste caso, o impacto emocional pode variar conforme a importância que o usuário dá à classificação. 
* **Ação Mitigadora:** Aprendi que, para garantir a equidade, é fundamental filtrar os dados de treinamento através de poses, expressões faciais e cores de roupas padronizadas para evitar erros de classificação baseados em variáveis irrelevantes.
